# Color Palette Generator

Generate beautiful, harmonious color palettes — pick a base color and get complementary, analogous, triadic, and split-complementary schemes instantly.

## Features

- Color harmony modes: Complementary, Analogous, Triadic, Tetradic, Split-Complementary, Monochromatic
- Shades and tints generator for each color
- HEX, RGB, HSL output with one-click copy
- Export palette as CSS custom properties, JSON, or PNG
- Accessibility contrast checker (WCAG AA / AAA)
