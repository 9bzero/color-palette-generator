# color-palette-generator

Generate color palettes from a base color.

## Modes

- **Complementary** — opposite on the color wheel
- **Analogous** — three adjacent colors
- **Triadic** — three equally spaced
- **Split-complementary** — two colors adjacent to the complement
- **Shades** — 10 tints and shades of the base color

Copy hex, RGB, or HSL values. Export the whole palette as CSS variables.

## Run

```bash
npm install && npm run dev
```
